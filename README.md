### 2019年12月9日 10点32分
添加远程访问ssh配置，避免每次登录都需要输入密码。  
具体方法：
首先在本地生成公钥和私钥。  
然后把公钥上传到服务器~/.ssh/下，新建文件authorized_keys  
将公钥写入进去。然后就可以无密码登录链接了。
